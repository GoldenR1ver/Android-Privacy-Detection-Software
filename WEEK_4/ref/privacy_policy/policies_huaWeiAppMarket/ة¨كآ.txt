扫呗APP应用（以下简称"本应用"）是由武汉利楚商务服务有限公司为您提供的移动收银管理助手。本应用尊重并保护所有使用服务用户的个人隐私权。为了给您提供更准确、更有个性化的服务，本应用会按照本隐私权政策的规定使用和披露您的个人信息。但本应用将以高度的勤勉、审慎义务对待这些信息。除本隐私权政策另有规定外，在未征得您事先许可的情况下，本应用不会将这些信息对外披露或向第三方提供。本应用会不时更新本隐私权政策。您在同意本应用服务使用协议之时，即视为您已经同意本隐私权政策全部内容。本隐私权政策属于本应用服务使用协议不可分割的一部分。
1.适用范围
) 在您注册本应用账号时,您根据本应用要求提供的个人注册信息.
) 在您使用本应用网络服务，或访问本应用平台网页时，本应用自动接收并记录的您的浏览器和手机.上的信息，包括但不限于您的IP地址、浏览器的类型、使用的语言、访问日期和时间、软硬件特征信息及您需求的网页记录等数据.
) 本应用通过合法途径从商业伙伴处取得的用户个人数据。
) 您在使用本应用平台提供的搜索服务时输入的关键字信息；
) 本应用收集到的您在本应用发布的有关信息数据，包括但不限于参与活动、成交信息及评价详情；
) 违反法律规定或违反本应用规则行为及本应用已对您采取的措施。
信息使用
) 本应用不会向任何无关第三方提供、出售、出租、分享或交易您的个人信息，除非事先得到您的许可，或该第三方和本应用（含本应用关联公司）单独或共同为您提供服务，且在该服务结束后，其将被禁止访问包括其以前能够访问的所有这些资料。
) 本应用亦不允许任何第三方以任何手段收集、编辑、出售或者无偿传播您的个人信息。任何本应用平台用户如从事上述活动，一经发现，本应用有权立即终止与该用户的服务协议。
) 为服务用户的目的，本应用可能通过使用您的个人信息，向您提供您感兴趣的信息，包括但不限于向您发出产品和服务信息，或者与本应用合作伙伴共享信息以便他们向您发送有关其产品和服务的信息（后者需要您的事先同意）。如果您介意我们使用您的信息来提供这些内容，您可以通过联系客服或者拨打我们的客服电话来取消个性化推送服务。
我们收集的信息
) 当您使用本应用服务时，为保障您正常使用我们的服务，维护我们服务的正常运行，改进及优化我们的服务体验以及保障您的账号安全，我们会收集您的设备型号、操作系统、唯一设备标识符（指由设备制造商编入到设备中的一串字符，可用于以独有方式标识相应设备）、登录IP地址、本应用软件版本号、设备MAC地址、BSSID、SSID、ICCID、Android ID、idfv和imsi、接入网络的方式、操作日志、服务日志信息等，这类信息是为提供服务必须收集的基础信息。我们还需要获取软件安装列表，以便判断当前状态下，应用是否正在前台运行。针对部分机型，我们会尝试写入SDCard数据(创建)和读取SDCard数据或写入/读取SD卡(sdcard)中的内容以及收集SD卡数据，用于保证设备在我们系统的唯一性。
) 您可以选择向我们提供您的指纹特征值信息，该信息属于敏感信息，拒绝提供该信息仅会使您无法使用指纹锁功能，但不影响您正常使用本应用的其他功能。
) 当您使用门店引流、会员管理、消费分期等功能时，您填写的活动名称、活动时间、门店地址等信息会存储在我们的服务器，因为存储是实现这一功能所必需的，我们会以加密的方式存储。
) 当您使用收款单功能时，我们会收集您以及您分享出去打开收款单界面的用户填写付款人姓名、联系电话、地址等信息。该信息属于敏感信息，拒绝提供这些信息仅会使您无法使用本应用收款单功能，但不影响您正常使用本应用的其他功能。
) 当您使用商户设置功能，我们将会收集您商户联系人信息、商户地址、商户手机号、商户邮箱、到账银行卡、身份证、营业执照、商户照片等信息。上述信息属于敏感信息，拒绝提供该信息仅会使您无法使用商户信息修改功能，但不影响您使用本应用的其他功能。
) 当您在使用在线客服功能，我们将会收集您录音和位置权限。上述信息属于敏感信息，拒绝提供该信息仅会使您无法使用商户信息修改功能，但不影响您使用本应用的其他功能。
) 鉴于结算账户所属银行会不定期开展商户营销活动,如需要参加系列营销活动,我司将会向银行提供相关信息(包括法人信息、经营信息及结算信息等与收单业务相关的内容）,用于评估是否符合活动要求。对营销活动期间相关数据进行风控管理、核对账目等。同意并授权：1）银行从我司接收并处理前述相关信息；2）银行向我司传输前述相关信息。我司与银行间均会按照国家法律法规对以上信息采取安全保护措施。若不同意前述授权，可向我司反馈取消授权。
) 当您在本应用使用第三方提供的服务时，第三方可能会获取您的昵称、头像、位置、姓名、证件类型、证件号码、手机号码信息；在经过您的明示同意后，第三方可获取您的以上信息；对于您在使用第三方提供的服务时主动提供给第三方的相关信息，我司将视为您允许该第三方获取上述此类信息；对于您在使用该第三方服务时产生的信息，应由您与该第三方依法约定上述信息的收集和使用事项。如您拒绝第三方在提供服务时收集、使用或者传递上述信息，将可能会导致您无法在电子银行中使用第三方的相应服务，但这不影响您使用本应用的其他功能。本应用内嵌和使用第三方SDK详见《扫呗APP第三方SDK目录》。
) 为了保证您可以正常接受到支付推送、统计账单及其他重要消息，避免消息遗漏，我们使用了App自启动和关联启动其他App功能；我们会通过客户端【通知】功能向您推送平台通知（如交易信息、资金变动、认证信息等），您可以在扫呗App我的-通知设置进行消息推送的设置，可选择关闭/接收。

) 如您出现违反中国有关法律、法规或者本应用服务协议或相关规则的情况，需要向第三方披露；
