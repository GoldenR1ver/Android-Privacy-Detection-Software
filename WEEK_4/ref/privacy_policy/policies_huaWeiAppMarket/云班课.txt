定义
我们：是指北京智启蓝墨信息技术有限公司，及由上述公司开发运营的云班课 App 软件。
个人信息（出自于 GB/T 35273-2017 《信息安全技术个人信息安全规范》）：是指以电子或者其他方式记录的能够单独或者与其他信息结合识别特定自然人身份或者反映特定自然人活动情况的各种信息。本隐私政策中涉及的个人信息包括自然人的基本资料（包括姓名、出生年份、性别、电话号码、电子邮箱）、网络身份标识信息（包括系统账号、邮箱地址以及与前述有关的密码）、个人财产信息（包括虚拟货币、虚拟交易）、个人上网记录（包括网站浏览记录、软件使用记录、点击记录）、个人常用设备信息（包括硬件序列号、硬件型号、设备 MAC 地址、操作系统类型、唯一设备标识符）、个人位置信息（包括大概地理位置、精确定位信息）。我们实际具体收集的个人信息种类以下文描述为准。
个人敏感信息（出自于 GB/T 35273-2017 《信息安全技术个人信息安全规范》）：是指一旦泄露、非法提供或滥用可能危害人身和财产安全，极易导致个人名誉、身心健康受到损害或歧视性待遇等的个人信息。本隐私政策中涉及的个人敏感信息包括您的个人财产信息、网络身份标识信息、精准定位信息。我们实际具体收集的个人敏感信息种类以下文描述为准。
Cookie ：Cookie 是包含字符串的小文件，在您登入和使用网站或其他网络内容时发送、存放在您的计算机、移动设备或其他装置内（通常经过加密）。Cookie 同类技术是可用于与 Cookie 类似用途的其他技术，例如：Web Beacon、Proxy、嵌入式脚本。
账号：当您注册账号并向我们提供一些个人信息，您就可以更好地使用我们的服务。当您访问云班课时，系统会利用这些账号信息对您进行身份验证，以防止未经授权的人员访问您的账号。
我们如何收集和使用您的信息
云班课依据法律法规以及遵循正当、合法、必要的原则而收集和使用您在使用服务过程中主动提供或由于产品和服务需要而产生的个人信息。如果我们欲将您的个人信息用于本隐私政策未载明的其它用途，或基于特定目的将收集而来的信息用于其他目的，我们会及时以合理的方式向您告知，并在使用前再次征得您的同意。
我们收集和使用的您的个人信息类型包括两种：第一种：我们产品与服务的核心业务功能所必需的信息：此类信息为产品与服务正常运行的必备信息，您须授权我们收集。如您拒绝提供，您将无法正常使用我们的产品与服务；第二种：附加业务功能可能需要收集的信息：此信息为非核心业务功能所需的信息，您可以选择是否授权我们收集。如您拒绝提供，将导致附加业务功能无法实现或无法达到我们拟达到的效果，但不影响您对核心业务功能的正常使用。
我们会为实现下述各项功能，收集和使用您的个人信息：
注册账号：当您注册云班课账号时，我们会收集您的手机号码，用于完成注册并保护您的账号安全。手机号码属于敏感信息，为了获得更高效、可靠的登录凭证，我们将收集您的手机号码。您可以在注册完成后自行决定是否需要完善您的头像、姓名、昵称、性别、出生年份、学校、院系、身份、工号/学号信息。如果您完善以上额外信息，将有助于我们为您提供更好的服务和体验。在上述额外信息中，如果您没有完善姓名和学号/工号信息，将会影响部分功能的使用，影响的功能包括：创建班课、加入班课。除此之外的其他信息若不提供，不会影响使用产品的基本功能。
当您使用云班课充值购买蓝豆（虚拟货币）时，我们会收集您的充值记录、消费记录信息，以便您查询自己的交易记录，同时尽最大程度保护您的财产、虚拟财产安全。上述信息属于敏感信息，但收集上述信息为实现相关功能所必须，否则将无法完成交易。
基于相机/摄像头的附加服务：当您使用资源和活动拍照上传功能、更换头像、更换班课封面、扫码投屏、私聊拍照发送图片、课程圈拍照发布日志功能时，我们会请求您授权相机权限。相机权限授权为实现上述功能所必要的，取消授权不影响您在云班课其他功能的正常使用。
基于相册（图片库/视频库）的图片/视频访问及上传的附加服务：当您使用更换头像、更换班课封面、课程圈发布图片日志、发布图片/视频类资源、在活动中提交图片/视频类附件、私聊发送图片/视频类消息功能时，我们会请求您授权相册权限。相册权限授权为实现上述功能所必要的，取消授权不影响您在云班课其他功能的正常使用。
基于麦克风语音技术的附加服务：当您使用录制音频以上传资源/附件、发送语音消息、快捷语音控制、课堂表现语音评分、小蓝/小墨对话功能时，我们会请求您授权麦克风权限。麦克风权限授权为实现上述功能所必要的，取消授权不影响您在云班课其他功能的正常使用。 
基于文件读取/存储（SD 卡读取/存储权限）的图片/视频/本地数据库存储的附加服务：当您使用从文件上传资源/附件、将图片/视频资源保存到相册功能时，我们会请求您授权 SD 卡读取/存储权限，当您允许授权后，云班课可以将网络数据缓存到您的本地数据库中，以节省数据流量。SD 卡读取/存储权限授权为实现上述功能所必要的，取消授权不影响您在云班课其他功能的正常使用。
当您在使用过程中出现程序崩溃时，为了优化您的使用体验，我们会采集并上报您的崩溃日志，iOS 端会采集设备型号、系统版本信息，Android 设备中会由腾讯 Bugly （第三方）收集和反馈您的崩溃信息，Bugly 会采集您的设备信息。
基于地理位置信息的服务：当您使用签到或者设置学校功能时，我们会请求您授权地理位置权限，在设置学校功能中，我们会将您的地理位置信息传递给高德地图 SDK 用于 POI 搜索，以获取您附近的学校信息。如果您是 Android 用户，在使用签到功能时，高德地图 SDK 会获取您的地理位置信息，仅用于计算发起签到的老师与参与签到的学生之间的距离。地理位置权限授权为实现上述功能所必要的，取消授权不影响您在云班课其他功能的正常使用。
我们会根据您加入或创建的班课所属课程，为您提供课程圈推荐，如果不希望接收课程圈推荐，您可以在【我的-设置-隐私设置】中关闭课程圈推荐。除取得您授权或法律法规另有规定外，我们不会将您的个人信息与您在应用程序中提供的任何个人身份信息相结合，以确保不识别到您个人。
我们向用户提供众多免费、优质的服务，为了保证良好的用户体验，尽可能地减少与您无关的广告对您造成的侵扰，我们会尽可能向您展示您可能更感兴趣、更实用的广告。在向您展示广告的过程中，我们将努力确保数据的安全性。如果不希望接收广告推荐，您可以在【我的-设置-隐私设置】中关闭广告推荐。

我们如何使用 Cookie
在您未拒绝接受 Cookie 的情况下，云班课会在您的计算机上设定或取用 Cookie，数据信息都采用匿名的方式，以便您能登录或使用依赖于 Cookie 的云班课平台服务或功能。
您有权选择接受或拒绝接受 Cookie 。您可以通过修改浏览器设置的方式拒绝接受 Cookie。但如果您选择拒绝接受 Cookie，您可能无法登录或使用依赖于 Cookie 的云班课网络服务或功能。
通过云班课所设 Cookie 所取得的有关信息，将适用本政策。
我们如何共享、转让、公开披露您的信息
共享
我们不会与任何第三方公司、组织和个人共享您的个人信息，除非获得您的明确同意。我们仅会处于合法、正当、必要、特定、明确的目的共享您的个人信息，并且只会共享提供服务所必要的个人信息。我们的合作伙伴无权将共享的个人信息用于任何其他用途，对我们与之共享个人信息的公司、组织和个人，我们会与其签署严格的保密协议，要求他们按照我们的说明、本隐私政策以及其他任何相关的保密和安全措施来处理个人信息，并且我们会要求合作伙伴提供数据安全能力和信息安全资质（等级保护测评、信息安全管理体系）说明。在个人敏感数据使用上，我们要求第三方采用数据脱敏加密技术，从而更好地保护用户数据。如果您对第三方使用您的个人信息有异议或发现这些第三方开发的应用程序存在风险时，我们建议您终止相关操作以保护您的合法权益，您还可以与相应的第三方协商解决，我们将尽可能提供帮助。如果我们在此过程中违反法律法规的规定，给您造成损失的，我们将承担因此引起的相应责任。
为实现本隐私政策中声明的功能，我们接入了第三方服务商提供的 SDK，并将我们依照本隐私政策收集的个人信息共享给第三方服务商，以完善服务和用户体验。云班课 App 涉及用户信息使用的 SDK 相关情况如下逐项列举：
高德地图提供的服务：我们集成高德地图 SDK 提供的服务，用于您在使用云班课时，基于您当前所在位置为您展示附近的学校功能。如果您是 Android 用户并使用签到功能时，高德地图 SDK 会为您提供定位服务。为实现高德地图 SDK 的基本服务功能，高德地图 SDK 需要收集如下信息，包括：位置信息（经纬度、精确位置、粗略位置），为了基于用户位置信息提供相应的选择学校服务；设备信息（IP 地址、GNSS 信息、WiFi 状态、WiFi 参数、WiFi 列表、SSID、BSSID、基站信息、信号强度的信息、蓝牙信息、传感器信息（矢量、加速度、压力、方向、地磁）、设备信号强度信息、外部存储目录），为了帮助开发者向用户提供定位、辅助定位，实现给开发者提供最终用户更精准的位置信息；设备标识信息（IMEI、IDFA、IDFV、Android ID、MEID、MAC 地址、OAID、IMSI、ICCID、硬件序列号）、当前应用信息（应用名、应用版本号）、设备参数及系统信息（系统属性、设备型号、操作系统、运营商信息），为了提供位置服务准确性和成功率，提供地图、搜索服务，追踪、排查、诊断、统计服务中的各类问题，保障产品和服务安全正常运行。高德地图 SDK 非常重视对您个人信息的保护，并采用业内通行的安全措施以确保信息安全。如果您不接受高德地图 SDK 服务且不希望您的个人信息被其处理和使用，您可登录高德开放平台官方网站 website_url_lnk 进行退出（opt-out）操作。为使您可以更好地了解高德地图 SDK 收集的类型及用途，以及保护个人信息的方式，您可以登录 website_url_lnk 进一步了解《高德地图开放平台隐私权政策》； 
友盟+ 提供的服务：您可以将云班课的内容分享到第三方平台，如微博、微信朋友圈、微信好友、QQ 空间、QQ 好友。在 iOS 端，我们集成友盟+ SDK，友盟+ SDK 需要收集您的设备标识符(IMEI/Mac/IDFA/OPENUDID/GUID、SIM 卡 IMSI 信息)，用于唯一标识设备，以便向目标设备分享消息。采集地理位置甄别分享通道，提供反作弊服务。如果您不希望友盟+ 采集这些数据，您可以通过访问 website_url_lnk 行使 opt-out 权利，一旦您行使 opt-out 权利，您的信息将不会被友盟+ 进行处理。为使您可以更好地了解友盟+ SDK 收集的类型及用途，以及保护个人信息的方式，您可以登录 website_url_lnk 进一步了解《友盟+ 隐私政策》； 
TalkingData 提供的服务：为了数据统计和分析目的，我们集成了由第三方 TalkingData（公司名称为：北京腾云天下科技有限公司） 提供的统计分析 SDK，我们需要向此 SDK 共享您的相关个人信息的目的、方式和范围。共享的个人信息类型与字段为：Android 设备品牌、型号、软件版本基础信息、应用列表信息、设备标识符（IMEI、MEID、SIM 卡 IMSI、ICCID、MAC、AndroidID、OAID、ADID）; iOS 设备品牌、型号、软件版本基础信息、设备标识符（IDFA、IDFV）、设备连接 WiFi 和基站信息、设备地理位置、SDK 宿主应用包名、版本号信息。其中，设备标识符用于生成脱敏的终端用户唯一标识，以提供基础分析能力；应用列表用于提供异常流量、虚假设备的识别以及反作弊能力；设备信息、网络信息、位置信息、应用信息均用于提供粗略的多维分析能力、分地区和应用版本号的数据报表。所有数据处理方式均采用数据加密技术对数据进行传输，采用去标识化方式对信息进行脱敏展示。为了您的信息安全，我们已与第三方 SDK 服务商签署严格的数据安全保密协议，该公司会严格遵守我们的数据隐私和安全要求。为使您可以更好地了解 TalkingData 收集的类型及用途，以及保护个人信息的方式，您可以登录 website_url_lnk 进一步了解《TalkingData 隐私政策》。同时我们理解并尊重您的选择权，如果您不愿参与 TalkingData 的大数据计算，您也可以通过 website_url_lnk 行使 opt-out 权力。您理解并同意，TalkingData 有权对已收集的数据进行去标识化和聚合性的处理后构建数据库，用以提供数据服务。如果 TalkingData SDK 收集使用个人信息的目的、方式、范围发生变化时，我们会以适当方式通知并提醒终端用户阅读； 
个推提供的服务：在 Android 设备中，我们使用由第三方个推消息推送 SDK 提供的消息推送技术服务用于推送/下发通知栏消息。在提供该服务的过程中，个推消息推送 SDK 需要收集您的相关个人信息，包括：设备信息（设备识别码、设备平台、设备厂商用于生成唯一的推送目标ID（CID）和设备ID（GID），以此保证消息推送的准确下发和消息设备的准确识别。设备品牌、设备型号及系统版本用于优化通道资源的有效性，提升消息推送的到达率和稳定性。此外，为确保CID的唯一性、提升消息推送稳定性和准确性，还需您授权存储权限（WRITE_EXTERNAL_STORAGE）和设备权限（READ_PHONE_STATE））、应用列表信息（个推采用合并链路技术进行技术推送，当一个设备有多个 App 的推送链路同时活跃时，个推会随机合并成一条链路，以达到为用户省电省流量的目的，因此需获得应用列表信息；同时个推提供智能推送功能，通过该信息推荐更符合您用户需要的推送内容，减少无用推送信息对您用户的打扰，您可以选择开启或关闭这项服务，但是关闭这项服务会导致电量和流量消耗加大）、网络信息以及位置相关信息（为了最大程度保持网络连接的稳定性，建立长链接，个推需要了解设备的网络状态和变化，从而实现稳定连续的推送服务。个推提供应景推送功能，位置相关信息将有助于个推为您提供线下场景的精细化推送，可以为您推荐更符合您用户需要的推送内容，减少无用推送消息对您用户的打扰）。 如果您需要行使删除权、更改授权同意的范围等，您可以查看《个推隐私政策》，或联系用户信息保护负责人邮箱 support@getui.com 的方式，向我们提出删除信息的请求。
