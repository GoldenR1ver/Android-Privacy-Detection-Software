账号的注册、登录与管理 
为了遵守法律法规关于网络实名制的要求，在您注册国铁吉讯账号时，我们需要识别您的身份。当您注册成为我们的用户时，您至少需要提供您的手机号。如果您通过注册 12306 账号成为我们的用户，您至少需要提供您的姓名、证件号、手机号；如果您通过登录您的 12306 账号或其他第三方账号成为我们的用户，您将授权我们从 12306 或第三方获取您的相关个人信息。
在成为我们的用户之后，您还可以选择向我们提供您的头像、昵称、生日、性别以及绑定您的 12306 账号或其他第三方账号，以便我们为您提供更便捷的服务以及更好的预防互联网欺诈。
安全管理
为了保障您正常使用我们提供的产品和服务，维护产品或者服务的安全运行，改进及优化我们的服务体验以及保障您的账号安全，我们将收集和使用您的设备信息，如设备型号、操作系统类型及版本、分辨率、电信运营商、设备 MAC 地址、唯一设备识别码（如 IMEI/Android ID/IDFA/OPENUDID/GUID 、 SIM 卡 IMSI 信息）、接入网络的方式、登录 IP 地址、账号信息、操作日志和服务日志等日志信息。
高铁 WiFi 服务
为了在高铁上向您提供 WiFi 服务，我们将收集您的手机号、账号信息和设备信息（SSID和BSSID），并扫描您的WiFi列表，以使您的设备顺利连接我们的 WiFi 网络。
行程服务和会员积分
为了向您提供行程服务和会员积分服务，我们将收集您的手机号、账号信息、 12306 账号、证件号、行程信息和位置信息。如您使用我们的 12306 行程同步功能，我们还需要对您的身份进行验证，以确保您的账号安全。您可以选择使用您的面部特征信息完成验证，我们仅接收验证结果，并不收集您的面部特征信息。如您不想使用面部特征验证，仍可通过其他方式录入您的行程。

（五） 我们如何使用 Cookie 和其他技术
Cookie 的使用
为了保证我们的产品或者服务的正常运转，我们会在您的设备上存储名为 Cookie 的小数据文件。在您后续访问我们的产品或者服务时， Cookie 可以帮助我们调用您既存的个人信息，以便您使用一键登录功能，获得更流畅的访问体验。
我们不会将 Cookie 用于本隐私政策所述目的之外的任何用途。您可根据自己的偏好管理或删除 Cookie 。请您注意，在清除 Cookie 之后，您将无法使用依赖于 Cookie 的功能。除 Cookie 外，我们还可能使用 Beacon 等业内通用的其他同类技术来收集您的个人信息。
SDK 的使用
为了向用户提供更为多元化的服务、保障我们客户端的稳定运行、功能实现，我们的应用中会嵌入第三方提供的软件工具开发包（ SDK ）、应用程序接口（ API ）或其他类似的应用程序，我们接入的第三方 SDK 详情可参考附件一《国铁吉讯使用 SDK 情况表》。我们将要求第三方对您的个人信息采取保护措施，并且严格遵守相关法律法规与监管的要求。此外，为了向用户提供更多服务、保证服务的稳定性和质量，或者升级相关功能时，我们可能会不时调整我们接入的第三方 SDK ，并对《国铁吉讯使用 SDK 情况表》进行适当更新。
自启动和关联启动说明
我们不会自动启动应用，但当您点击广告后，一些广告会关联启动第三方APP，不点击广告不会关联启动，或 者我们提示您是否可以给个好评时，若你点击给好评，会关联启动第三方应用市场让您方便写好评，不点击给好评不会关联启动。我们的服务若出现系统崩溃导致退 出，会出现重新启动应用的情况，以方便用户继续使用。
